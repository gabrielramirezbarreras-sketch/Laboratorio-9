Practica 1: Instalacion y acceso a webmin
Paso 1: Actualizar el sistema.
sudo apt update

Paso 2: Instalar dependencias.
sudo apt install wget apt-transport-https software-properties-common -y

Paso 3: Descargar Webmin.
wget https://www.webmin.com/download/deb/webmin-current.deb

Paso 4: Instalar Webmin.
sudo dpkg -i webmin-current.deb

Si hay errores de dependencias:
sudo apt --fix-broken install -y

Acceso a Webmin:
https://localhost:10000

Credenciales:
Usuario: tu usuario de Linux
Contraseña: tu contraseña

Practica 2 Desplieque de una VM con terraform en digital Ocean.
sudo apt update && sudo apt install -y gnupg software-properties-common curl

curl -fsSL https://apt.releases.hashicorp.com/gpg | sudo apt-key add -

sudo apt-add-repository "deb https://apt.releases.hashicorp.com $(lsb_release -cs) main"

sudo apt update && sudo apt install terraform -y

terraform -version

sudo apt install awscli -y

aws configure

Te pedirá:

AWS Access Key
AWS Secret Key
Región (ejemplo: us-east-1)
Output format: json

sudo nano main.tf

Configuración básica:
provider "aws" {
  region = "us-east-1"
}

resource "aws_key_pair" "deployer" {
  key_name   = "ansible-key"
  public_key = file("~/.ssh/id_rsa.pub")
}

resource "aws_instance" "OS3vm" {
  ami           = "ami-0c02fb55956c7d316" # Ubuntu 22.04 (puede cambiar según región)
  instance_type = "t2.micro"

  key_name = aws_key_pair.deployer.key_name

  tags = {
    Name = "OS3vm"
  }
}

terraform init

terraform validate

terraform plan

terraform apply

terraform show

Practica 3: Instalación de Ansible.
Instalar Ansible:
sudo apt update
sudo apt install ansible -y

Verifica:
ansible --version

Instalar dependencias necesarias:
sudo apt install sshpass openssh-client -y

Crear usuario "ansible" (en el controller):
sudo adduser ansible

Agregar a sudoers:
sudo usermod -aG sudo ansible

Cambiar a ese usuario:
su - ansible

Configurar acceso SSH sin contraseña:
ssh-keygen -t rsa -b 2048

Copiar clave:

ssh-copy-id ansible@192.168.140.129

Crear usuario ansible también:
sudo adduser ansible
sudo usermod -aG sudo ansible

Crear Directorio:
sudo mkdir -p /etc/ansible

Editar:
sudo nano /etc/ansible/hosts

Agregar:
[linux]
linux1 ansible_host=192.168.140.129 ansible_user=ansible

[win]
windows1 ansible_host=192.168.140.1 ansible_user=ansible ansible_connection=ssh

Probar conexión con ping:
ansible linux -m ping

Practica 4: Comandos Ad-Hoc.
Crear Direcotrio:
mkdir -p /home/ansible/Desktop

Poner informacion:
echo "Prueba Ansible" > ~/Desktop/archivo.txt

Verifica que existe:
ls ~/Desktop/archivo.txt

Créalo de nuevo correctamente
nano /home/ansible/hosts

PEGA:
[linux]
linux1 ansible_host=192.168.140.129 ansible_user=ansible

Prueba inventario:
ansible-inventory -i /home/ansible/hosts --list

Comando correcto:
ansible linux -i /home/ansible/hosts -m ping -u ansible --ask-pass

Reiniciar la VM con reboot:
ansible linux -i ./hosts -m reboot -b -K

Practica 5: Playbooks:
sudo nano instalar_notepad.yml

Contenido:
- name: Instalar Notepad++ en Windows
  hosts: windows
  gather_facts: no

  tasks:
    - name: Instalar Notepad++ usando Chocolatey
      win_chocolatey:
        name: notepadplusplus
        state: present

Ejecuta en PowerShell como administrador:

Set-ExecutionPolicy Bypass -Scope Process -Force
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.SecurityProtocolType]::Tls12
iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

Playbook para instalar NGINX:
nano instalar_nginx.yml

Crear archivo:
sudo nano instalar_nginx.yml

Contenido:
- name: Instalar y levantar NGINX
  hosts: linux
  become: yes

  tasks:
    - name: Actualizar repositorios
      apt:
        update_cache: yes

    - name: Instalar NGINX
      apt:
        name: nginx
        state: present

    - name: Iniciar servicio NGINX
      service:
        name: nginx
        state: started
        enabled: yes

Inventario:
[linux]
localhost ansible_connection=local

Ejecutar:
ansible-playbook -i hosts instalar_nginx.yml

Validación:
systemctl status nginx

O en navegador:

http://localhost
